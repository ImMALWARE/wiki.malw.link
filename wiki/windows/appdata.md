# Папка AppData в Windows

**AppData** — это скрытая системная папка в Windows, в которой программы хранят свои настройки, временные файлы, кэш и данные профиля пользователя. По умолчанию она находится по пути `C:\Users\Имя_Пользователя\AppData`.

Внутри неё находятся директории, выполняющие разные задачи:

## Roaming (%appdata%)

Используется для хранения важных данных программ, которые могут синхронизироваться (перемещаться) между разными компьютерами в корпоративной сети. Даже на домашних компьютерах большинство программ сохраняют свои конфигурации, профили браузеров и сохранения игр именно сюда.

## Local (%localappdata%)

В папке **Local** лежат данные, привязанные строго к одному конкретному компьютеру: тяжелый кэш, скачанные обновления и временные файлы. Эти данные не синхронизируются по сети из-за их большого объема. Некоторые программы (например, Discord или браузеры) устанавливаются в Local целиком, чтобы обходить требование прав администратора при установке.

## Как открыть эти папки

Поскольку папка AppData по умолчанию скрыта, проще всего открывать нужные директории через системные переменные:

*   Для открытия **Roaming** — вызовите окно [Выполнить](/windows/run) (`Win + R`), введите `%appdata%` и нажмите Enter.
*   Для открытия **Local** — в окне Выполнить введите `%localappdata%`.
